BookFolio turns the bookmarks already in your Chrome profile into a
navigation system woven through the browser: a new tab dashboard, a side
panel, omnibox search, and a right-click "Save to BookFolio" menu — with
visual organization through colors, folder emoji, and grid or list views.

Reads only what's already in your browser profile. No accounts, no sync
layer of its own, no copy stored anywhere, no network calls.

## Features

- **New tab dashboard** — your full bookmark folder hierarchy in a
  collapsible sidebar, with a searchable grid view.
- **Side panel** — compact bookmark navigation and search, plus a "From this
  site" section for the active tab's domain. Open it from the toolbar popup
  or a keyboard shortcut.
- **Command palette** — `Ctrl+Shift+F` (`Cmd+Shift+F` on macOS) opens the
  side panel with search focused, from anywhere in Chrome.
- **Omnibox search** — type `bf` in the address bar to search bookmarks
  without opening a tab.
- **Save to BookFolio** — right-click a page or a link to bookmark it
  straight into any folder from a menu that mirrors your folder tree.
- **Colors, emoji & groups** — color-code bookmarks and folders, add emoji
  to folders, and create named groups you can open all at once.
- **Pinned & recents strips** — quick access to favorites and recently
  visited bookmarks.
- **Open Tabs & Recently Closed panels** — see your open tabs and restore
  recently closed ones, read locally from Chrome's own session list.
- **Per-site badge** — the toolbar icon shows how many bookmarks you have
  for the current site.
- **Light / dark / auto themes**, plus custom dashboard backgrounds.
- **Full keyboard navigation** throughout the dashboard and side panel.

## Privacy

BookFolio reads only the bookmarks and tabs already in your Chrome profile.
No accounts. No tracking. No data leaves your device.

See the full [Privacy Policy](privacy.md) for details on permissions and
locally stored data.
